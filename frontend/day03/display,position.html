📐 display / position / z-index / visibility 예제

1. display

display: block (한 줄 차지)
display: inline (가로만) display: inline-block (inline + 크기 조절)
display: none (보이지 않음)

2. position

position: static (기본값)
position: relative (자기 위치 기준 이동)
position: absolute (부모 기준)
position: fixed (스크롤해도 고정)

3. z-index

z-index: 1
z-index: 2 (위에 올라감)

4. visibility

visibility: visible (기본값)
visibility: hidden (공간 차지하지만 안 보임)
